# KernelCharDriver
Linux Character Device Driver implementation using kernel modules with open, read, write and ioctl operations.

# Linux Character Device Driver

This project implements a simple Linux Character Device Driver using Kernel Modules.  
The driver allows communication between user space and kernel space using standard system calls.

The project demonstrates core concepts of Linux device driver development and system programming.

---

## Features

- Loadable Kernel Module (LKM)
- Character Device Driver
- open() system call
- read() system call
- write() system call
- ioctl() support
- Kernel ↔ User space communication

---

## Requirements

- Linux Operating System
- GCC Compiler
- Kernel Headers Installed
- Root/Sudo access
